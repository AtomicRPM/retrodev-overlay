# RetroDev Overlay

The RetroDev Overlay provides Gentoo packages for retro-computing development.

Current goals:

- Z88DK
- SJASMPlus
- XC3SPROG
- ACME
- cc65
- PASMO
- Fuse
- VICE
- OpenFPGALoader

Long-term goals include complete development environments for:

- ZX Spectrum
- Commodore 64
- Commodore 128
- VIC-20
- MSX
- RC2014
- FPGA/CPLD
